    ## 关于这本书

    - **作者**: 尼尔·埃亚尔、瑞安·胡佛
    - **出版年**: 2017
    - **一句话主旨**: 用触发、行动、多变酬赏和投入四步检查产品是否能形成健康、可持续且合乎伦理的使用习惯。
    - **整书理解**: [BOOK_OVERVIEW.md](BOOK_OVERVIEW.md)

    ## Skill 列表

    | Skill | 中文名 | 什么时候用 |
    |---|---|---|
    | [`habit-zone-assessment`](habit-zone-assessment/SKILL.md) | 习惯区间评估 | 当用户想判断一个产品是否适合做成习惯型产品，或只是低频工具、一次性服务时调用。 |
| [`trigger-design-map`](trigger-design-map/SKILL.md) | 触发设计地图 | 当用户需要设计外部触发并连接用户内部情绪触发时调用。 |
| [`action-ability-simplifier`](action-ability-simplifier/SKILL.md) | 行动能力简化器 | 当用户希望提高某个关键行为发生率，需要降低时间、金钱、体力、脑力、社会偏差或非常规性成本时调用。 |
| [`variable-reward-selector`](variable-reward-selector/SKILL.md) | 多变酬赏选择器 | 当用户想设计能持续吸引用戶的回报机制，需要区分社交、猎物和自我酬赏时调用。 |
| [`investment-value-loop`](investment-value-loop/SKILL.md) | 投入价值循环 | 当用户想让产品随着用户使用而更有价值，并把一次使用连接到下一次触发时调用。 |
| [`manipulation-ethics-screen`](manipulation-ethics-screen/SKILL.md) | 操控伦理筛查 | 当团队使用习惯设计但担心产品会变成操控、剥削或伤害用户时调用。 |
