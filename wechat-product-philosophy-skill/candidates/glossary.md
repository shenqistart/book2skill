# 《微信背后的产品观》术语候选

> 来源说明见 [../source/SOURCE.md](../source/SOURCE.md)。以下为方法论术语重述。

| 术语 | 方法论含义 | 相关 skill |
|---|---|---|
| 用户 | 有惰性、情绪、欲望、关系和环境约束的具体人 | `human-group-sensing` |
| 群体 | 用户之间相互影响后形成的整体行为 | `human-group-sensing` |
| 需求 | 心理诉求在特定场景中的功能化表达 | `demand-psychology-mining` |
| 场景 | 环境、关系、设备、任务和心理状态的组合 | `scenario-boundary-design` |
| 产品结构 | 功能之间的骨架、分类、抽象和演化关系 | `product-structure-evolution` |
| 产品 DNA | 长期决定取舍的价值观和认知底层 | `product-structure-evolution` |
| 自然增长 | 产品价值驱动的自发扩散与留存 | `natural-growth-judgment` |
| 气质 | 文案、视觉、默认行为、隐私和打扰方式共同表达出的产品人格 | `product-spirit-expression` |
