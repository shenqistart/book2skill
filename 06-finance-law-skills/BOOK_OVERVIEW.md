## 1. 结构

### 类型

多书方法论分类：公司金融、金融经济学、应用公司理财、创业投资法律。

### 一句话主旨

这组书共同回答：企业如何在价值创造、风险定价、融资结构、现金分配和法律契约约束之间做可解释的资本决策。

### 骨架

1. **问题分流层**：达摩达兰把公司理财压缩为投资、融资、现金返还三类问题，帮助 agent 先判断问题入口。
2. **项目价值层**：布雷利/迈尔斯/艾伦与达摩达兰共同强调项目收益、现金流、最低可接受收益率和 NPV 纪律。
3. **风险定价层**：徐高和公司金融教材提供 CAPM、均值方差、无套利、风险中性定价等理论背景。
4. **融资结构层**：公司金融教材把 WACC、MM、税盾、APV 和再平衡假设连接成融资影响价值的检查框架。
5. **估值与分配层**：达摩达兰把投资、融资、股利政策和 DCF/相对估值连接，要求假设一致。
6. **法律契约层**：创业投资法律手册把公司设立、股权、章程、知识产权、投资协议和治理风险放回中国法律环境。

**论点关系**：达摩达兰给出 CFO/经营者的问题地图；布雷利等提供公司金融教材骨架；徐高解释资产定价与金融摩擦的理论来源；法律手册给出中国创业公司把资本决策落地为合法契约的边界。

### 核心问题

- 一个项目、融资方案、估值结论或股权安排，究竟是在创造价值，还是只是在移动风险、隐藏假设或转嫁法律责任？
- 当财务模型和法律契约冲突时，应该先查哪类事实、哪组假设、哪条边界？

## 2. 解释

### 关键术语

| 术语 | 本分类中的用法 | 易混点 |
|---|---|---|
| 投资决策 | 把资源投向项目、资产、并购、信用政策或运营改造的选择 | 不只等于买股票 |
| 最低可接受收益率 | 项目必须超过的风险调整后回报门槛 | 不是老板拍脑袋的目标收益 |
| NPV | 未来现金流现值减去初始投入后的价值增量 | 不是会计利润 |
| 资本机会成本 | 同风险替代投资要求的回报 | 不是融资利率本身 |
| WACC | 按资本结构加权的整体资本成本 | 只有在假设成立时才适合贴现 |
| APV | 基础项目价值加融资副效应现值 | 不是另一个万能估值公式 |
| CAPM | 用系统性风险解释预期收益的资产定价模型 | 不是任何风险都要补偿 |
| 无套利 | 相同现金流不应有不同价格的定价约束 | 需要市场可交易、可复制等前提 |
| 金融摩擦 | 信息不对称、有限套利、非理性、交易成本等模型失效来源 | 不是公式错了的万能借口 |
| 公司章程 | 股东权利、治理结构和公司运行规则的基础文件 | 不应只用模板应付登记 |
| 资本充实 | 股东出资、注册资本和债务责任之间的法律约束 | 不等于注册资本越大越好 |
| 股权安排 | 股权比例、出资、转让、激励、控制权和退出机制的组合 | 不只是“谁占多少” |

### 核心命题

1. 财务问题首先要分流：是投资、融资、现金返还、估值还是法律契约问题。
2. 投资项目必须用增量现金流和风险调整后的最低可接受收益率检验。
3. 风险不是越大越应该高估值；只有不可分散、与市场相关或无法转移的风险才应进入资本成本。
4. WACC 可用但不无条件成立；资本结构重大变化、固定负债或融资副效应复杂时，应转向 APV。
5. 估值不是套公式，而是让现金流、增长、风险、再投资和融资假设保持一致。
6. 现金返还政策要看公司是否还有足够好的投资项目，而不是只看账上现金多不多。
7. 金融理论必须同时检查均衡、无套利和摩擦；模型前提不满足时，要先说明失效点。
8. 创业融资和公司治理必须回到法律文件：章程、出资、股权、投资协议、知识产权和劳动合规会决定资本决策能否落地。

### 论证链

先用达摩达兰的问题地图确认资本决策类型，再用公司金融的 NPV、资本成本和估值框架展开，随后用金融经济学检查资产定价前提与摩擦，最后用创业法律手册把财务方案翻译为可执行、可登记、可治理、可追责的法律结构。

## 3. 批判

### 作者的时代局限

- 公司金融教材和达摩达兰原书的案例、税制、资本市场环境主要来自欧美语境，不能直接替代中国当前制度。
- 《创业投资法律手册》基于 2015 年前后的法律环境，部分法律、监管和司法口径可能已变化。
- 《金融经济学二十五讲》是导论性讲义，强调理论主线，不负责覆盖全部实务细节。

### 作者的立场盲点

- 公司金融教材偏资本市场和大公司视角，可能低估早期创业公司的信息不透明、控制权和融资可得性问题。
- 金融经济学模型偏理论一致性，可能低估交易限制、监管干预和行为偏差。
- 法律问答书偏风险规避与合规提醒，可能低估创业速度、谈判力量和商业试错的现实压力。

### 未被证明的假设

- 市场价格、可比公司和资本成本能提供足够可靠的风险信号。
- 管理层会以股东价值最大化为目标，而不是受代理问题、控制权或短期压力影响。
- 合同条款只要设计充分就能有效执行，但现实中还受证据、诉讼成本和监管解释影响。

### 最强反对意见

这组书横跨财务模型、金融理论和中国法律环境，若不先识别问题类型和适用边界，agent 容易把“估值公式 + 法律常识 + 教材概念”拼成看似完整却不可执行的建议。
