# 候选术语提取

| 术语 | 分类内定义 | 主要来源 |
|---|---|---|
| 投资决策 | 企业把资源投入项目、资产或经营行动的选择 | 达摩达兰、布雷利等 |
| 融资决策 | 企业选择债务、股权、混合证券或内部现金支持投资的方式 | 达摩达兰、布雷利等 |
| 股利决策 | 企业决定向所有者返还多少现金及返还形式 | 达摩达兰 |
| 最低可接受收益率 | 项目必须超过的风险调整后收益门槛 | 达摩达兰 |
| NPV | 预期现金流现值减去投资额后的净价值 | 布雷利等、达摩达兰 |
| 资本成本 | 资本提供者基于风险要求的回报 | 布雷利等 |
| WACC | 按债务和股权市场价值权重计算的加权平均资本成本 | 布雷利等 |
| APV | 基础项目 NPV 加融资副效应现值 | 布雷利等 |
| CAPM | 用系统性风险 beta 解释预期收益的模型 | 徐高、达摩达兰 |
| 无套利 | 相同未来现金流应有相同价格的定价约束 | 徐高 |
| 金融摩擦 | 信息不对称、有限套利、行为偏差、交易成本等模型前提缺口 | 徐高 |
| 公司章程 | 公司治理、股东权利和决策机制的基础法律文件 | 杨春宝、王成兵 |
| 出资责任 | 股东按章程和法律履行出资及相关责任 | 杨春宝、王成兵 |
| 知识产权保护 | 著作权、商标、域名、商号、专利、技术秘密等组合保护 | 杨春宝、王成兵 |
