# Framework Candidates

## F01 founder-mission-value-system

- **type**: framework
- **source_chapter**: 《小米创业思考》前言、第一章；《经营管理全集》第 5 卷；《段永平投资问答录》
- **source_quote**:
  > "商业的目的是什么，如何让商业实现最大化的现实意义？我的答案是：效率。"
- **summary**: 把创业决策放回使命、用户价值、社会贡献和不做清单中，先定义企业长期存在的理由，再决定业务、产品和增长动作。
- **tags**: [mission, values, founder, long-term]

## F02 nonconsensus-monopoly-thesis

- **type**: framework
- **source_chapter**: 《从 0 到 1》第 3 章、第 8 章
- **source_quote**:
  > "这里的垄断企业指的仅仅是一种企业，它供给消费者的产品其他企业无法供给。"
- **summary**: 从非共识秘密、市场定义和防守结构判断一个机会是否可能从竞争红海进入独特供给。
- **tags**: [opportunity, monopoly, nonconsensus]

## F03 disruptive-entry-finder

- **type**: framework
- **source_chapter**: 《创新者的解答》第 2 章、第 3 章；《小米创业思考》第一章
- **source_quote**:
  > "客户是在雇用产品去完成某项特别的任务。"
- **summary**: 用客户任务、新市场破坏和低端市场破坏寻找大公司不愿意或不能有效服务的切入口。
- **tags**: [disruption, jobs-to-be-done, entry]

## F04 validated-learning-loop

- **type**: framework
- **source_chapter**: 《精益创业》验证式学习、开发-测量-认知循环；《精益创业实战》
- **source_quote**:
  > "成功的创业过程是加速这个反馈循环的过程。"
- **summary**: 把创业项目拆成假设，通过 MVP 和真实行为指标产生有效学习，再决定坚持、调整或转向。
- **tags**: [lean-startup, mvp, validation]

## F05 lean-business-model-risk-map

- **type**: framework
- **source_chapter**: 《精益创业实战》精益画布；《创业之殇》市场、产品、执行和资金部分
- **source_quote**:
  > "最好的方法是先把商业模式写在一页纸上。"
- **summary**: 用一页商业模式图定位最大风险，避免先写漂亮计划书，再用融资故事掩盖问题。
- **tags**: [business-model, risk, lean-canvas]

## F06 user-frontline-flywheel

- **type**: framework
- **source_chapter**: 《小米创业思考》第八章、第九章；《富甲美国》第 12 章；《一胜九败》第一章
- **source_quote**:
  > "如果能够开一家不怎么需要店员服务，又能卖好衣服的休闲服店，这不是更好吗？"
- **summary**: 从用户现场和前线反馈开始，形成产品改进、口碑、成本效率和复购的飞轮。
- **tags**: [customer, frontline, flywheel]

## F07 manager-output-leverage

- **type**: framework
- **source_chapter**: 《格鲁夫给经理人的第一课》经理人的产出；《关乎天下》领袖管理篇
- **source_quote**:
  > "经理人的产出=他直接管辖部门的产出+他间接影响所及部门的产出。"
- **summary**: 把管理者从个人工作者改写为杠杆设计者，通过会议、培训、决策和信息流放大团队产出。
- **tags**: [management, leverage, output]

## F08 talent-culture-operating-system

- **type**: framework
- **source_chapter**: 《重新定义公司》创意精英；《小米创业思考》组建团队；《经营者养成笔记》
- **source_quote**:
  > "一定要找有共同的愿景、有能力、有责任心的人。"
- **summary**: 用人才标准、文化约束、责任下放和经营者培养机制，让组织不靠创始人单点推动。
- **tags**: [culture, talent, operating-system]

## F09 hard-things-wartime-leadership

- **type**: framework
- **source_chapter**: 《创业维艰》前言、裁员和战时 CEO 章节；《一胜九败》失败章节
- **source_quote**:
  > "真正的难题并不是设置一个宏伟目标，而是你在没有实现目标时不得不忍痛裁员。"
- **summary**: 公司进入生死压力时，领导者要直面坏消息、现金约束、裁员沟通和组织士气，而不是继续讲愿景。
- **tags**: [wartime, crisis, leadership]

## F10 failure-forensics-map

- **type**: framework
- **source_chapter**: 《大败局》序言；《十亿美金的教训》前言；《创业之殇》
- **source_quote**:
  > "我们发现其中最为致命的是：中国许多企业家的体内潜伏和滋生着一种共同的失败基因。"
- **summary**: 用失败链条复盘市场错判、模式悖论、资本冒进、创始人英雄主义、组织失灵和现金断裂。
- **tags**: [failure, forensics, risk]
