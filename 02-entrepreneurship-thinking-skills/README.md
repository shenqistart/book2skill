# 02 创业思维 Skills

本目录把行者创业系统读书产品中 `02_创业思维` 分类下的成功企业家方法论、从 0 到 1、运营管理和失败教训类书籍，蒸馏成一组可被 agent 调用的创业判断 skills。

它不是创业故事合集，也不是融资、法律或投资建议。它的用途是：当用户面对创业方向、商业模式、验证路径、组织管理、艰难决策或失败复盘时，先选对思维工具，再按步骤拆解。

## 推荐调用方式

1. 先读 [INDEX.md](./INDEX.md)，按用户问题选择 skill。
2. 如果问题是“为什么创业、为谁创造价值、哪些事不该做”，优先用 `founder-mission-value-system`。
3. 如果问题是机会判断，先用 `nonconsensus-monopoly-thesis` 或 `disruptive-entry-finder`。
4. 如果问题是早期验证和商业模式，使用 `validated-learning-loop` 与 `lean-business-model-risk-map`。
5. 如果问题是增长、用户、门店或一线运营，使用 `user-frontline-flywheel`。
6. 如果问题进入组织管理，使用 `manager-output-leverage` 与 `talent-culture-operating-system`。
7. 如果公司已经遇到现金、裁员、转型或生死问题，使用 `hard-things-wartime-leadership`，再用 `failure-forensics-map` 做复盘。

## 重要边界

- 不提供证券投资、融资交易、法律或税务结论。
- 不把成功企业家的故事当作可复制公式。
- 不把精益验证当成永远小步试错；涉及强监管、硬件供应链和长期技术突破时必须补充行业约束。
- 不提交原书全文、图片或大段原文；源文本只用于本地方法论审计。
