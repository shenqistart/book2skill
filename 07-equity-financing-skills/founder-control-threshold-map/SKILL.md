# 创始人控制权阈值地图

## R — 原文 (Reading)

> “67%的持股比例”
>
> 来源: 李利威《一本书看透股权架构》第1章。

## I — 方法论骨架 (Interpretation)

这个 skill 把“谁控制公司”从一个持股百分比问题, 改写成一张分层权限地图。

第一层是股权比例: 67%、51%、34%、20%、10% 等数字分别对应不同强度的表决、阻断、提案或救济能力。第二层是公司章程和特别约定: 默认规则可能被章程、股东协议、表决权委托、一致行动安排改写。第三层是董事会与经营权: 股东会权力不等于日常经营权。第四层是公司类型和司法门槛: 有限责任公司、股份有限公司、上市公司、小股东救济门槛并不相同。

它的核心判断不是“某人是不是绝对控股”, 而是: 这张 cap table 在重大事项、普通经营、融资、退出和冲突升级时会不会卡死。

## A1 — 书中的应用 (Past Application)

### 案例 1: A/B/C 三股东的控制错觉

- **问题**: A 持股 51%, B 持股 41%, C 持股 8%, A 以为自己已经控制公司。
- **方法论的使用**: 先检查重大事项阈值, 再看小股东救济和解散门槛。
- **结论**: 51% 可能能通过普通事项, 但不一定能单独通过需要更高比例的重大事项。
- **结果**: 发现 C 的 8% 在某些争议场景下既可能影响僵局, 又可能达不到独立救济门槛。

### 案例 2: 34% 的阻断线

- **问题**: 创始人融资后低于控股比例, 担心被投资人完全左右。
- **方法论的使用**: 检查是否仍保留重大事项阻断能力。
- **结论**: 34% 附近不是经营控制权, 但常被用作重大事项否决线。
- **结果**: 融资谈判从“估值越高越好”转向“稀释后还剩哪些权力”。

### 案例 3: 股东会与董事会错位

- **问题**: 股权比例看似合理, 但董事会席位安排使经营决策被另一方主导。
- **方法论的使用**: 同时画股东会表决权、董事会席位和经营授权。
- **结论**: 股权优势不能自动推出经营优势。
- **结果**: 在股权比例之外补审董事提名、董事会表决、CEO 任免和预算批准条款。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. 用户问“这样分股谁说了算”“51% 是否够控制公司”“34% 有什么用”。
2. 用户准备融资、增资、引入员工持股或合伙人, 需要看稀释后的权力变化。
3. 用户已经出现股东分歧, 需要判断哪些事项能通过、哪些事项会卡住。
4. 用户只给出 cap table, 但没有说明章程、董事会和股东协议。

### 语言信号

- “我还有没有控制权?”
- “创始人至少要留多少股?”
- “投资人拿 20% 会不会影响控制?”
- “股东僵局怎么提前看出来?”

### 与相邻 skill 的区分

- 与 `control-without-ownership-design` 的区别: 本 skill 先判断权力边界; 后者设计钱权分离工具。
- 与 `term-sheet-economics-control-audit` 的区别: 本 skill 可用于任何分股结构; 后者聚焦融资条款清单。

## E — 可执行步骤 (Execution)

当 skill 被激活后, agent 应按以下步骤执行:

1. **收集权力材料**
   - 完成标准: 列出股权比例、表决权比例、董事会席位、章程特别约定、股东协议和已签委托/一致行动安排。
   - 判停条件: 只有股权比例时, 先声明结果只是初步风险图。

2. **标出阈值线**
   - 完成标准: 标出 67%、51%、34%、20%、10% 等关键线, 并说明它们在当前公司类型和章程下是否适用。

3. **拆分事项类型**
   - 完成标准: 至少区分普通经营、重大资产/融资、章程修改、董事任免、利润分配、公司解散或清算。

4. **画出通过权与阻断权**
   - 完成标准: 对每类事项分别写明“谁能单独通过、谁能联合通过、谁能阻断、谁无法有效救济”。

5. **输出僵局和补救建议**
   - 完成标准: 给出 3 类风险: 控制权过散、控制权过独、权利和责任不匹配; 同时提示需律师复核的条款。

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 用户要求具体法律结论、诉讼胜率、税务筹划或上市合规判断。
- 用户的问题主要是员工激励工具设计, 应转 `control-without-ownership-design`。
- 用户已经拿到 Term Sheet, 需要逐项审条款, 应转 `term-sheet-economics-control-audit`。

### 作者在书中警告的失败模式

- 把 51% 误认为完整控制权。
- 只看钱的比例, 不看表决权、董事会和章程。
- 让小股东既没有足够权利, 又承担无法退出的冲突成本。

### 作者的盲点 / 时代局限

- 阈值判断高度依赖当时有效的法律、公司类型和章程约定。
- 本 skill 只能做结构检查, 不能替代律师对协议文本的解释。

### 容易混淆的邻近方法论

- “估值稀释”: 它回答钱变少多少; 本 skill 回答权力变到哪里。
- “合伙人贡献分配”: 它回答如何分配收益; 本 skill 回答分配后如何治理。
