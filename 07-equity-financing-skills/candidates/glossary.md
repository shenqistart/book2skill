# Glossary Candidates

| 术语 | 来源 | 本轮解释 |
|---|---|---|
| 股东捣蛋线 | 《一本书看透股权架构》 | 有限公司中约 34% 表决权可阻断需三分之二以上通过的重大事项 |
| 完美控制线 | 《一本书看透股权架构》 | 有限公司中约 67% 表决权才接近完整控制重大事项 |
| 持股平台 | 《一本书看透股权架构》 | 将员工、创始人或投资人权益集中到有限合伙/公司等主体中持有核心公司股权 |
| 期权池 | 《风险投资交易》 | 为员工和顾问预留的股权激励池, 融资前设置会影响真实投资前估值 |
| 完全稀释 | 《风险投资交易》 | 把期权、认股权证、优先股转换等潜在股份全部计入后的股权口径 |
| 投资前估值 | 《风险投资交易》 | 投资者出资前公司的估值; 与投资后估值混淆会改变投资人拿股比例 |
| 投资后估值 | 《风险投资交易》 | 投资前估值加投资额后的估值 |
| 经济因素 | 《风险投资交易》 | 决定投资人从退出或清算中获得多少收益的条款 |
| 控制因素 | 《风险投资交易》 | 决定投资人否决权、董事会权利和重大事项控制的条款 |
| Term Sheet | 《风险投资交易》《创业之初你不可不知的融资知识》 | 投资主要条款清单, 是交易蓝图和谈判里程碑 |
| 尽职调查 | 《创业之初你不可不知的融资知识》 | 投资人对市场、业务、人员、财务、法律等进行全方位核查 |
| 大数法则 | 《超级天使投资》 | 早期投资单个项目不可预测, 但足够多、足够长期的组合可接近期望结果 |
