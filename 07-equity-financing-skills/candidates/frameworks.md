# Framework Candidates

## F01 创始人控制权阈值地图

- **来源**: 《一本书看透股权架构》第1章, 《股权战争》序言和阿里巴巴/雷士案例。
- **方法单元**: 把股权比例、表决权、董事会席位和公司类型放到一张控制权地图里, 区分 10%、34%、51%、67%、30% 等阈值的实际含义。
- **候选用途**: 判断创始人、合伙人、投资人或员工持股安排是否会制造否决权、僵局或控制权旁落。

## F02 分股不分权工具选择框架

- **来源**: 《一本书看透股权架构》第2章。
- **方法单元**: 在需要共享经济利益但集中治理权时, 比较有限合伙、金字塔、一致行动、委托投票、章程控制、优先股、AB 股。
- **候选用途**: 员工激励、联合创始人分股、财务投资人进入、上市前持股平台设计。

## F03 融资准备度与 runway 框架

- **来源**: 《穿越寒冬》第一章/第二章, 《给你一个亿你能干什么》财务预测和融资计划, 《创业之初你不可不知的融资知识》商业计划书与融资流程。
- **方法单元**: 在融资前检查客户证据、资金用途、财务预测、最低 runway、资料准备和创始人自己对融资的掌控度。
- **候选用途**: 判断现在是否应该融资, 该融多少钱, 是否已经过早/过晚。

## F04 投资人匹配信号过滤器

- **来源**: 《穿越寒冬》风险投资内幕和投资人类型, 《超级天使投资》天使投资人特征, 《创业之初你不可不知的融资知识》VC 兴趣点。
- **方法单元**: 按阶段、基金规模、退出需求、资源匹配、跟投能力、战略冲突和声誉过滤投资人。
- **候选用途**: 多个投资人/机构可选时选择优先沟通顺序, 或判断某笔钱是否会带来结构性副作用。

## F05 估值稀释澄清框架

- **来源**: 《风险投资交易》第4章, 《创业之初你不可不知的融资知识》第5章, 《给你一个亿你能干什么》财务预测段落。
- **方法单元**: 把投资前估值、投资后估值、完全稀释、期权池、认股权证、下一轮融资和真实创始人持股一起算清。
- **候选用途**: 用户被估值数字吸引但不清楚真实稀释时。

## F06 Term Sheet 经济/控制条款审计

- **来源**: 《风险投资交易》第3-6章, 《创业之初你不可不知的融资知识》Term Sheet/DD 段落, 《股权战争》资本局案例。
- **方法单元**: 把 Term Sheet 分为经济因素和控制因素, 逐项检查清算优先、防稀释、董事会、保护性条款、领售/共售、对赌、回购和创始人兑现。
- **候选用途**: 收到投资意向书、准备谈判、复核条款风险。

## F07 天使投资组合纪律

- **来源**: 《超级天使投资》第1章投资组合理论。
- **方法单元**: 把天使投资当作高失败率组合游戏, 用 20-80 家、单笔额度、合格投资人、筛选漏斗和长期参与约束投资冲动。
- **候选用途**: 新手天使、个人投资人、创业者理解天使投资人行为。

## F08 股权战争预演

- **来源**: 《股权战争》序言和案例体系, 《一本书看透股权架构》股东僵局案例, 《风险投资交易》董事会/控制条款。
- **方法单元**: 在融资或分股前预演资本方、联合创始人、亲属股东、董事会、监管和退出路径的冲突剧本。
- **候选用途**: 合伙协议、融资条款、控制权安排签署前的风险审查。
