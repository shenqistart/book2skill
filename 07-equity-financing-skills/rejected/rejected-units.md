# Rejected Units

## R01 创业者要有勇气

- **来源**: 《穿越寒冬》《给你一个亿你能干什么》
- **拒绝原因**: V3 不通过。勇气、坚持、承担风险是常识性原则, 不能形成精准可调用 skill。
- **保留方式**: 可作为融资准备度 skill 的语气背景, 不独立成 skill。

## R02 创业者要会讲故事

- **来源**: 《穿越寒冬》路演和故事段落, 《给你一个亿你能干什么》创业表达。
- **拒绝原因**: V1/V3 不充分。讲故事很重要, 但本轮源书没有形成区别于常见 pitch skill 的独特步骤。
- **保留方式**: 并入 `financing-readiness-runway` 的材料准备检查。

## R03 中国 VC 和硅谷 VC 差异

- **来源**: 《穿越寒冬》相关章节。
- **拒绝原因**: V2 不足。更像背景比较, 不适合直接变成执行 skill; 且实时市场差异已经变化。
- **保留方式**: 放入边界提醒: 当前融资市场和地域差异需另行查证。

## R04 VIE 科普

- **来源**: 《股权战争》阿里巴巴案例附文。
- **拒绝原因**: 高法律/监管风险, 时效性强, 单独做 skill 容易被误用为法律意见。
- **保留方式**: 并入 `equity-conflict-pre-mortem` 的监管/控制协议风险项。

## R05 股权众筹平台选择

- **来源**: 《超级天使投资》众筹和平台章节。
- **拒绝原因**: 时效性强, 规则变化快, 且用户未要求当前平台推荐。
- **保留方式**: 作为 `angel-portfolio-discipline` 的边界: 涉及当前平台和法规时需实时核验。
