# 缠论中枢定位

## R — 来源

依据《教你炒股票17：走势终完美》和《教你炒股票20：走势中枢级别扩张及第三类买卖点》。

## I — 方法论骨架

中枢是缠论判断走势类型的核心。
一个走势是否是盘整、趋势、延伸或扩张，不看主观感觉，而看次级别走势之间是否形成重叠区间。
完成的走势类型至少包含一个中枢；趋势通常要求同向、同级别中枢之间不重叠。
如果两个同级别中枢周围波动区间发生重叠，就可能形成更大级别中枢。
因此，中枢定位的目标不是“找支撑压力”，而是判断走势结构的状态。

## A1 — 书中的应用

### 案例: 日线与30分钟级别差异

- **问题**: 类似图形为什么有的算日线级别走势，有的只算 30 分钟级别走势。
- **方法论的使用**: 通过是否在对应级别形成中枢来区分。
- **结论**: 走势级别由中枢结构决定，不由肉眼相似度决定。
- **结果**: 避免把小级别走势误判成大级别趋势。

## A2 — 触发场景

1. 用户问“这里是中枢延伸还是新中枢”。
2. 用户给出几段次级别走势，要求计算中枢区间。
3. 用户想判断当下是盘整、趋势还是级别扩张。

### 语言信号

- “中枢区间是多少”
- “这是中枢扩张吗”
- “这里还是趋势吗”
- “有没有形成更大级别中枢”

### 与相邻 skill 的区分

- 与 `chanlun-structure-parser` 的区别: 本 skill 假设已有笔/线段或次级别走势。
- 与 `chanlun-buy-sell-points` 的区别: 本 skill 判断地图，不直接判断买卖点。

## E — 可执行步骤

1. **确认分析级别与次级别**
   - 完成标准: 明确“当前级别”和构成它的“次级别走势”。

2. **列出连续三段次级别走势**
   - 完成标准: 至少有三段可比较区间；不足则停止。

3. **计算重叠区间**
   - 完成标准: 给出中枢下沿、上沿，并说明计算依据。

4. **判断中枢状态**
   - 完成标准: 输出“延伸 / 新生 / 扩张 / 趋势延续 / 无法确认”之一。

5. **给出失效条件**
   - 完成标准: 明确什么价格行为会推翻当前中枢判断。

## B — 边界

- 没有次级别走势段时不要硬算中枢。
- 不要把普通箱体震荡直接等同于缠论中枢。
- 不要把“突破中枢”自动当成买点，仍需看回试和级别。

### 失败模式

- 只看价格区间，不看构成中枢的走势段。
- 混用不同级别的数据。
- 把中枢扩张误判为趋势延续。

### 盲点 / 时代局限

- 低流动性和跳空较多的品种，次级别走势可能不连续。
- 当下未完成走势会影响中枢边界确认。
