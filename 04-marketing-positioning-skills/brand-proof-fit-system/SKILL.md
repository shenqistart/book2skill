# 品牌证据与配称系统

## R — 原文 (Reading)

> “品牌三问：你是什么、有何不同、何以见得。”
>
> “配称与商业模式。”
>
> 来源：《升级定位》第7-11章。

## I — 方法论骨架 (Interpretation)

一个定位方案只有在回答三件事时才算进入可执行状态：你是什么，有何不同，何以见得。

“你是什么”让用户知道如何分类。
“有何不同”让用户知道为什么不选别人。
“何以见得”让用户相信差异不是自夸。
但证据还不够，产品体验、价格、渠道、组织能力、服务承诺和商业模式都要支持同一个定位。
如果传播说高端，渠道像低价清仓；如果声称专业，服务无法兑现；如果主张健康，成分和证据不足，定位就会崩。

这个 skill 的核心是把定位从表达审查推进到经营一致性审查。

## A1 — 书中的应用 (Past Application)

### 案例 1：品牌三问

- **问题**：企业常把品牌定位说成抽象愿景或情绪词。
- **方法论的使用**：用“你是什么、有何不同、何以见得”拆成品类、差异和证据。
- **结论**：无法回答任一问题，定位就不完整。
- **结果**：品牌方案从好听变成可审计。

### 案例 2：配称与商业模式

- **问题**：定位常停留在广告和视觉层。
- **方法论的使用**：检查产品、渠道、价格和商业模式是否共同支撑定位。
- **结论**：定位必须变成经营活动的选择标准。
- **结果**：品牌战略进入组织执行。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. 用户已有定位语，但不知道是否可信。
2. 用户想做品牌升级、招商材料、官网首页或融资叙事。
3. 用户担心广告说法和产品真实体验不一致。
4. 用户需要把品牌战略落到产品、渠道、服务和组织动作。

### 语言信号

- “这个定位有什么问题？”
- “我们怎么证明这个差异？”
- “品牌主张和业务动作怎么对齐？”
- “用户凭什么相信我们？”

### 与相邻 skill 的区分

- 与 `mind-ladder-positioning` 的区别：先选心智位置，再用本 skill 查证据和配称。
- 与 `influence-defense-screen` 的区别：本 skill 检查可信证据；后者检查是否使用操纵性影响力。

## E — 可执行步骤 (Execution)

1. **拆出品牌三问**
   - 完成标准：分别写出“你是什么”“有何不同”“何以见得”。

2. **标注证据等级**
   - 完成标准：区分事实证据、用户证据、第三方证据、体验证据和主观声明。

3. **检查经营配称**
   - 完成标准：逐项检查产品、价格、渠道、传播、服务、组织和商业模式是否支持定位。

4. **找出断裂点**
   - 完成标准：列出至少 3 个最可能让用户不信的断点。

5. **给出修正路径**
   - 完成标准：对每个断点给出“补证据、改承诺、改体验、改渠道或放弃定位”的建议。

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 品牌还没有明确目标用户和心智位置。
- 用户只要求创意发想，不需要战略审查。
- 涉及医疗、金融、法律等高监管声明时，需要专业合规审核。

### 作者在书中警告的失败模式

- 只有差异，没有证据。
- 只有广告，没有配称。
- 只看内部愿景，不看外部心智。

### 作者的盲点 / 时代局限

- 证据标准在不同行业差异很大，不能用消费品品牌证据标准直接套 B2B、医疗或金融。
- 平台评价、达人背书和用户生成内容会改变证据呈现方式。
