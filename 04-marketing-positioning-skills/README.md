# 04 营销和定位 Skills

本目录把 `04_营销和定位` 下的定位、口碑、符号、行为改变、传播、原型、文化战略和消费时代书籍，蒸馏成一组可被 agent 调用的营销判断 skills。

它不是营销话术库，也不是广告创意模板。它的用途是：当用户面对品牌定位、传播扩散、购买行为、文化趋势或消费者时代变化问题时，先选对分析框架，再按步骤拆解。

## 推荐调用方式

1. 先读 [INDEX.md](./INDEX.md)，按用户问题选择 skill。
2. 如果问题是“品牌到底占据什么心智位置”，从 `mind-ladder-positioning` 开始。
3. 如果问题是“能不能开新品类或重新定义赛道”，使用 `category-entry-design`。
4. 如果问题是“品牌定位是否可信、能否落到经营配称”，使用 `brand-proof-fit-system`。
5. 如果问题是“如何做口碑、社群、用户共创”，使用 `participation-word-of-mouth-loop`。
6. 如果问题是“怎样让品牌更容易被记住”，使用 `super-symbol-memory-anchor`。
7. 如果问题是“如何改变消费者行为或防止操纵”，使用 `behavior-change-lever-stack` 与 `influence-defense-screen`。
8. 如果问题是“内容如何疯传或流行如何引爆”，使用 `contagious-trigger-design` 与 `diffusion-tipping-diagnosis`。
9. 如果问题是“品牌意义、原型、文化趋势和时代大势”，使用 `archetype-meaning-system` 与 `cultural-era-opportunity-map`。

## 重要边界

- 不生成欺骗性营销、虚假背书、暗黑模式或操控用户的方案。
- 不把品牌定位当成一句口号；定位必须回到品类、心智、证据和配称。
- 不把传播扩散当成单纯流量技巧；需要检查产品价值、传播载体和伦理边界。
- 不把美国、日本或中国品牌案例直接机械套用；涉及当前市场数据、政策和平台规则时需要另行查证。
- 不复制原书大段内容，源文本只用于方法论审计。
