# 文化时代机会地图

## R — 原文 (Reading)

> “意识形态机遇。”
>
> “购物使人幸福时代的终结。”
>
> 来源：《文化战略》；《第四消费时代》序言。

## I — 方法论骨架 (Interpretation)

有些品牌机会不是来自功能更好，而是来自旧文化正统已经无法解释用户生活。

文化战略要求先识别主流品牌都在模仿什么文化套路，再观察社会变化、亚文化实践和用户矛盾中是否出现新的意义需求。
第四消费时代提醒我们，消费偏好会从大量拥有、个性炫耀转向共享、地方、关系、简约和生活意义。
品牌如果仍用旧时代语言说服新用户，就会显得空洞。
但文化机会必须被产品、价格、渠道、组织和真实行为承载，不能只贴热点标签。

这个 skill 的核心是把“做趋势营销”改写成“识别社会变化中的可承载品牌意义”。

## A1 — 书中的应用 (Past Application)

### 案例 1：文化战略中的意识形态机遇

- **问题**：成熟市场中的品牌常陷入相似功能和相似价值点。
- **方法论的使用**：从历史、社会、亚文化和意识形态变化中寻找旧正统失灵之处。
- **结论**：突破性品牌创新来自新的文化表达。
- **结果**：品牌差异不只是产品属性，也可以是新的社会意义。

### 案例 2：第四消费时代

- **问题**：日本消费社会从物质拥有和个性追求转向新的幸福问题。
- **方法论的使用**：把消费分为阶段，观察人口、城市、家庭、地方、共享和关系变化。
- **结论**：用户对“拥有”的需求会下降，对连接、地方和生活意义的需求会上升。
- **结果**：品牌判断必须放入消费社会阶段。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. 用户想判断品牌是否符合消费趋势和社会大势。
2. 用户的功能差异很弱，但可能有文化表达机会。
3. 用户面对地方、共享、社区、可持续、简约生活等消费变化。
4. 用户想避免蹭热点，寻找可长期承载的品牌意义。

### 语言信号

- “这个品牌是否符合大势？”
- “消费降级/消费分化下该怎么做品牌？”
- “我们能不能从文化趋势里找机会？”
- “这个品牌是不是还停留在旧消费时代？”

### 与相邻 skill 的区分

- 与 `archetype-meaning-system` 的区别：本 skill 看社会文化变化；原型 skill 看心理欲望和品牌角色。
- 与 `diffusion-tipping-diagnosis` 的区别：本 skill 判断文化机会；后者判断扩散临界。

## E — 可执行步骤 (Execution)

1. **识别文化正统**
   - 完成标准：列出当前品类主流品牌都在重复的价值点、形象和话语。

2. **寻找社会矛盾**
   - 完成标准：找出用户生活中新的压力、身份冲突、地方变化、代际变化或消费观变化。

3. **观察亚文化和边缘实践**
   - 完成标准：列出已经在小圈层出现的新表达、新用法或新生活方式。

4. **映射消费时代**
   - 完成标准：判断目标用户更偏大量拥有、个性炫耀、共享关系、地方意义或混合状态。

5. **设计可承载表达**
   - 完成标准：把文化机会落到产品、渠道、服务、内容、社群和组织行为。

6. **排除蹭热点**
   - 完成标准：说明品牌为什么有资格表达这个文化机会，缺什么则先补什么。

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 用户要实时趋势预测、市场规模测算或投放热点追踪。
- 品牌没有能力承载文化主张，只想贴标签。
- 文化表达可能挪用、冒犯或简化弱势群体经验。

### 作者在书中警告的失败模式

- 传统营销把复杂文化语境简化成“天然、健康、活力”等空泛价值点。
- 大公司品牌官僚体系忽略亚文化和历史变化。
- 把消费社会阶段当线性必然，而不是具体市场假设。

### 作者的盲点 / 时代局限

- 《第四消费时代》基于日本经验，中国市场可能多阶段并存。
- 文化战略需要深度研究，不能用少量社媒观察替代田野、数据和历史分析。
