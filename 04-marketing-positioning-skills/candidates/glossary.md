# Glossary Candidates

| 术语 | 分类级定义 |
|---|---|
| 心智阶梯 | 用户在某个品类中已有的品牌排序和认知结构 |
| 定位 | 在顾客心智中占据清楚、可防守、可传播的位置 |
| 品类 | 顾客理解选择和比较时使用的分类入口 |
| 视觉锤 | 将品牌定位钉入心智的视觉符号 |
| 品牌三问 | 你是什么、有何不同、何以见得 |
| 配称 | 所有经营动作对定位的一致支撑 |
| 参与感 | 用户通过参与产品、品牌、活动、服务形成认同与传播 |
| 口碑铁三角 | 产品、社会化媒体和用户关系共同构成的口碑机制 |
| 超级符号 | 被大众已有文化记忆支持的高识别符号 |
| 行为杠杆 | 改变行为的动机、能力、触发、社会和反馈要素 |
| 影响力原则 | 互惠、承诺一致、社会认同、喜好、权威、稀缺等心理捷径 |
| STEPPS | 社交货币、诱因、情绪、公共性、实用价值、故事 |
| 引爆点 | 小变化触发大规模流行的临界点 |
| 连接员/内行/推销员 | 扩散网络中的关键人物类型 |
| 原型 | 品牌意义背后稳定的人类欲望和角色模式 |
| 意识形态机遇 | 社会历史变化导致旧文化正统失效后出现的品牌创新窗口 |
| 第四消费时代 | 重视共享、关系、地方、简约和生活意义的消费阶段 |
