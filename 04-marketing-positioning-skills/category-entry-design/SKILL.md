# 品类入口设计

## R — 原文 (Reading)

> “创建一个新品类的方法并没有那么昂贵。”
>
> “品牌是品类及其特性的代表。”
>
> 来源：《21世纪的定位》第5章；《升级定位》第6章。

## I — 方法论骨架 (Interpretation)

当顾客已经把旧品类的第一名、第二名记得很牢时，新品牌继续说“我更好”通常无效。

更好的问题是：顾客是否需要一个新的分类入口？
新品类不是随意造词，而是把用户已有问题、技术变化、渠道变化或生活方式变化压缩成一个更容易选择的名字。
好的品类入口要让顾客立刻知道“这是什么、和旧东西有什么不同、为什么现在需要它”。
如果品类名、品牌名、视觉锤和证据互相支撑，品牌就不必每次从零解释。

这个 skill 的核心是把竞争从旧赛道排名转为新入口设计。

## A1 — 书中的应用 (Past Application)

### 案例 1：21 世纪定位中的品类机会

- **问题**：全球化、城市化、互联网和技术变化让旧定位原则面临新环境。
- **方法论的使用**：从品类、品牌名、竞争对手、二元性、视觉锤等角度重新定义进入心智的方法。
- **结论**：企业应优先寻找新品类或新特性的代表位置。
- **结果**：定位从“广告传播”升级为“品类入口竞争”。

### 案例 2：升级定位中的品类三界

- **问题**：创业者常把产品品类、渠道品类和导购品类混在一起。
- **方法论的使用**：分辨用户如何购买、如何搜索、如何被导购和渠道呈现。
- **结论**：品类设计要匹配真实购买路径。
- **结果**：品类命名变成战略选择，而不是命名游戏。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. 用户说“这个市场太卷了，我们是不是要开新品类”。
2. 用户要给新产品、AI 工具、服务模式、课程或品牌命名。
3. 用户发现用户不知道该把产品归到哪类。
4. 用户想定义对立面，如“不是 X，而是 Y”。

### 语言信号

- “这算不算一个新品类？”
- “我们应该怎么命名这个赛道？”
- “用户不知道我们是什么。”
- “我们能不能不跟大品牌正面打？”

### 与相邻 skill 的区分

- 与 `mind-ladder-positioning` 的区别：先确认旧阶梯是否有位置；没有位置时再设计新品类入口。
- 与 `super-symbol-memory-anchor` 的区别：本 skill 负责品类和名字，后者负责符号记忆。

## E — 可执行步骤 (Execution)

1. **确认旧品类拥挤度**
   - 完成标准：列出旧品类第一梯队和用户为什么不会自然想到你。

2. **寻找变化源**
   - 完成标准：至少识别一个技术、渠道、行为、场景、政策、生活方式或价值观变化。

3. **设计品类候选**
   - 完成标准：提出 3 个以内候选品类名，每个都能回答“是什么”和“区别于谁”。

4. **测试可理解性**
   - 完成标准：非内部人员能在 10 秒内说出该品类解决什么问题。

5. **绑定证据和视觉锤**
   - 完成标准：为品类入口配置品牌名、关键证据、视觉符号或代表场景。

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 旧品类中已有清楚且可防守的位置。
- 产品差异不足，只想用新品类包装普通功能。
- 用户教育成本过高，且没有资源长期投入。

### 作者在书中警告的失败模式

- 品类名只服务内部战略，用户听不懂。
- 品牌名、品类名和视觉锤互相脱节。
- 为了显得创新而逃避真实竞争。

### 作者的盲点 / 时代局限

- 新品类能否成立高度依赖平台搜索、渠道货架和社交语境，不能只凭命名判断。
- 中国市场中导购、直播和平台分类可能强烈塑造品类入口。
