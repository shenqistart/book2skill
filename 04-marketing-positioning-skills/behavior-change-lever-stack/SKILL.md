# 行为改变杠杆栈

## R — 原文 (Reading)

> “改变消费者行为的十大策略。”
>
> “社会认同。”
>
> 来源：《如何让他买》；《影响力》。

## I — 方法论骨架 (Interpretation)

营销最终要改变某个具体行为，而不是只改变用户态度。

先把目标行为说清楚：购买、试用、分享、预约、到店、复购还是替换旧方案。
再找阻力：用户是不想做、不会做、太麻烦、不信任、没有触发、害怕风险，还是身边没人这么做。
行为杠杆可以分成五类：提高动机，降低能力门槛，设置触发，利用正当社会影响，提供即时反馈。
每个杠杆都要接受伦理检查，不能用欺骗和操纵换短期转化。

这个 skill 的核心是把“怎么说服用户”改写成“目标行为在哪个环节断了”。

## A1 — 书中的应用 (Past Application)

### 案例 1：从态度到行为

- **问题**：用户可能喜欢一个品牌，却仍然不购买。
- **方法论的使用**：把购买拆成目标行为、行为障碍和可干预杠杆。
- **结论**：营销要处理行动成本和触发时机。
- **结果**：转化问题不再只靠文案解决。

### 案例 2：社会认同

- **问题**：用户在不确定时会参考他人行为。
- **方法论的使用**：用真实的用户证据、评价和采用行为降低不确定性。
- **结论**：社会影响能推动行动，但必须真实透明。
- **结果**：影响力原则成为行为杠杆，也需要伦理边界。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. 用户说“用户有兴趣但不下单/不试用/不复购”。
2. 用户想提高到店、预约、提交表单、转介绍或切换旧方案。
3. 用户需要设计转化漏斗，但不清楚障碍在哪里。
4. 用户想把心理学原则用于正当转化。

### 语言信号

- “为什么用户不买？”
- “怎样提高试用转化？”
- “用户觉得不错但没有行动。”
- “怎么让用户从知道变成行动？”

### 与相邻 skill 的区分

- 与 `influence-defense-screen` 的区别：本 skill 设计行为杠杆；后者审查这些杠杆是否操纵。
- 与 `participation-word-of-mouth-loop` 的区别：本 skill 关注具体行为转化；后者关注长期参与和口碑。

## E — 可执行步骤 (Execution)

1. **定义目标行为**
   - 完成标准：写成“谁在什么场景下完成什么动作”。

2. **定位行为断点**
   - 完成标准：判断断点属于动机、能力、触发、信任、社会影响、风险或反馈。

3. **选择最小杠杆**
   - 完成标准：每个断点选择一个最小干预，如降低步骤、增加提醒、展示真实案例、试用保障。

4. **设计可测指标**
   - 完成标准：为每个杠杆定义可观察指标，如点击、预约、试用、复购、分享。

5. **做伦理筛查**
   - 完成标准：确认信息真实、选择自愿、退出清楚、风险透明。

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 产品价值未成立，只想靠心理技巧强推。
- 用户利益可能受损，且缺少充分告知。
- 目标行为涉及医疗、金融、法律等高风险选择。

### 作者在书中警告的失败模式

- 以为改变态度就等于改变行为。
- 忽略行动门槛和触发时机。
- 滥用社会认同、权威或稀缺制造压力。

### 作者的盲点 / 时代局限

- 行为心理案例常来自特定文化和实验环境，迁移到具体产品需要 A/B 测试和用户研究。
- 转化提升不能替代产品长期价值。
