# 候选方法论单元

## 通过候选

1. **走势结构识别**
   - 来源: 分型、笔、线段相关章节。
   - 价值: 为所有后续分析提供统一标注规则。
   - 通过原因: 跨章节反复出现，具备明确执行步骤，非普通技术分析常识。

2. **中枢定位**
   - 来源: 走势终完美、中枢级别扩张、第三类买卖点相关章节。
   - 价值: 判断盘整、趋势、延续、扩张的核心。
   - 通过原因: 是缠论术语系统的中心节点。

3. **背驰诊断**
   - 来源: MACD 对背驰的辅助判断、转折力度与级别相关章节。
   - 价值: 用于判断趋势力度衰竭和潜在转折。
   - 通过原因: 可转换为 A-B-C 段检查流程。

4. **三类买卖点识别**
   - 来源: 买卖点完备性、第三类买卖点相关章节。
   - 价值: 把结构分析转成交易计划检查。
   - 通过原因: 具备明确触发场景和边界。

5. **多级别情景推演**
   - 来源: 级别延续、背驰后转折分类相关章节。
   - 价值: 防止把小级别信号误解为大级别反转。
   - 通过原因: 直接降低误用风险。
