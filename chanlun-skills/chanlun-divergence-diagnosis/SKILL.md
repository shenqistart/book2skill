# 缠论背驰诊断

## R — 来源

依据《教你炒股票24：MACD对背驰的辅助判断》和《教你炒股票29：转折的力度与级别》。

## I — 方法论骨架

背驰不是“指标背离”四个字那么简单。
标准背驰需要在同一分析级别下，识别 A-B-C 三段结构。
B 段通常对应一个更大或同分析框架内的中枢，并使 MACD 回拉到 0 轴附近。
C 段与 A 段同向，但力度弱于 A 段，才构成候选背驰。
背驰只说明某级别趋势力量衰竭；之后可能是弱反弹、中枢扩张、更大级别盘整或反趋势。

## A1 — 书中的应用

### 案例: 用 MACD 辅助判断 A-B-C 段

- **问题**: 用户难以直接从中枢结构判断背驰。
- **方法论的使用**: 先划分 A-B-C，再比较 C 段与 A 段的 MACD 柱面积和力度。
- **结论**: MACD 可作为辅助，但不能替代中枢和级别。
- **结果**: 背驰判断从单指标感觉转成结构化检查。

## A2 — 触发场景

1. 用户问“这里是不是背驰/背离”。
2. 用户给出 A-B-C 段，希望判断 C 段力度是否弱。
3. 用户想知道背驰后可能是什么级别的转折。

### 语言信号

- “是不是顶背驰”
- “MACD红柱变短”
- “C段比A段弱吗”
- “背驰后会怎么走”

### 与相邻 skill 的区分

- 与 `chanlun-buy-sell-points` 的区别: 本 skill 判断背驰条件；买卖点类型要另行判断。
- 与 `chanlun-multi-level-scenario` 的区别: 本 skill 先给背驰诊断，多级别预案可在之后组合调用。

## E — 可执行步骤

1. **确认级别和 A-B-C 段**
   - 完成标准: 明确 A、B、C 的起止和方向。
   - 判停条件: 如果 A-B-C 不清晰，要求先调用结构/中枢 skill。

2. **检查 B 段中枢作用**
   - 完成标准: 判断 B 段是否承担连接、回拉或中枢功能。

3. **比较 A 与 C 的力度**
   - 完成标准: 从价格推进、斜率、成交量、MACD 柱面积等角度比较。

4. **区分趋势背驰与盘整背驰**
   - 完成标准: 输出“标准背驰 / 盘整背驰 / 不成立 / 数据不足”。

5. **输出后续情景**
   - 完成标准: 至少列出弱反弹、级别扩张、反趋势三类可能及确认信号。

## B — 边界

- 不要仅凭 MACD 柱缩短就断定背驰。
- 不要跨级别比较 A 与 C。
- 不要把背驰等同于马上大反转。

### 失败模式

- 先有主观看多/看空，再强行寻找背驰。
- 忽略 B 段中枢，只比较两个指标峰值。
- 把小级别背驰当作大级别转折。

### 盲点 / 时代局限

- MACD 参数、交易品种、波动结构不同，会影响辅助判断。
- 高频、跳空、涨跌停市场中，指标面积比较可能失真。
