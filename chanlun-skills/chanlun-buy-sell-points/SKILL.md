# 缠论三类买卖点识别

## R — 来源

依据《教你炒股票17：走势终完美》《教你炒股票20：走势中枢级别扩张及第三类买卖点》《教你炒股票21：买卖点分析的完备性》。

## I — 方法论骨架

三类买卖点是缠论把结构分析转成交易计划的接口。
第一类买卖点通常与趋势背驰后的转折相关。
第二类买卖点是第一类买卖点之后，第一次次级别回调或反抽形成的确认位置。
第三类买卖点是离开中枢后第一次回试不回到中枢的延续位置。
判断买卖点时必须先确认级别、中枢位置、离开/回试关系，最后再给成立和失效条件。

## A1 — 书中的应用

### 案例: 第三类买点

- **问题**: 突破中枢后，哪些回调可以视为延续机会。
- **方法论的使用**: 检查是否为次级别向上离开中枢后的第一次回试，且低点不跌回中枢。
- **结论**: 只有满足结构条件，才是候选第三类买点。
- **结果**: 避免把任何突破回踩都当成三买。

## A2 — 触发场景

1. 用户问“这里是一买、二买还是三买”。
2. 用户问“这里突破回踩算不算第三类买点”。
3. 用户想让 AI 输出交易计划的成立和失效条件。

### 语言信号

- “一买”
- “二买”
- “三买”
- “三卖”
- “回踩不回中枢”
- “不要给建议，只判断结构”

### 与相邻 skill 的区分

- 与 `chanlun-divergence-diagnosis` 的区别: 一买/一卖常需要背驰诊断，但本 skill 负责买卖点分类。
- 与 `chanlun-zhongshu-mapper` 的区别: 中枢是前置地图，本 skill 判断地图上的位置。

## E — 可执行步骤

1. **声明非投资建议**
   - 完成标准: 输出只做结构判断，不给买入/卖出指令。

2. **确认级别和中枢**
   - 完成标准: 明确分析级别、中枢区间、离开方向。

3. **匹配买卖点类型**
   - 完成标准: 按以下规则输出候选类型:
     - 趋势背驰后转折: 候选一买/一卖。
     - 一买/一卖后的次级别回调/反抽: 候选二买/二卖。
     - 离开中枢后第一次回试不回中枢: 候选三买/三卖。

4. **列出成立条件**
   - 完成标准: 给出需要继续确认的结构条件。

5. **列出失效条件**
   - 完成标准: 给出跌回/升回中枢、破坏次级别结构等无效条件。

## B — 边界

- 不输出“现在买入/卖出”。
- 不在没有中枢和级别信息时硬判买卖点。
- 不把第三类买点后续必然趋势化；它也可能演化为更大级别盘整。

### 失败模式

- 把普通回踩当成三买。
- 把任何指标背离当成一买。
- 忽略二买与三买可能重合但条件严格。

### 盲点 / 时代局限

- 实盘成交、仓位、滑点和止损不在原始结构规则内，需要交易系统另行处理。
- 不同交易品种的涨跌停、流动性会改变可执行性。
