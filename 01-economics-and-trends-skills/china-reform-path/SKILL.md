# 中国改革路径与制度红利识别

## R — 原文 (Reading)

> “供给侧结构性改革是基础性制度的改革、体制性机制性的改革。”
>
> “市场的大闸被小心翼翼地打开的时候，自由的水流就开始渗透了进来。”
>
> 来源：黄奇帆《分析与思考》；吴晓波《激荡三十年》。

## I — 方法论骨架 (Interpretation)

中国改革分析要看“制度阀门”如何打开。

很多变化不是单个企业突然变强，而是产权、准入、资本、土地、户籍、开放、监管和地方试验的约束发生变化。
同样的商业模式，在不同阶段会有完全不同的合法性、融资能力和增长边界。
因此判断中国大势时，要先问：哪个制度约束被松动，哪些要素被重新配置，谁获得新激励，谁承担调整成本。

这个 skill 把政策和企业故事还原成改革路径。

## A1 — 书中的应用 (Past Application)

### 案例 1：个体经济和民营萌芽

- **问题**：改革开放早期民营经济如何出现。
- **方法论的使用**：吴晓波记录个体户、家庭副业、地方试验和意识形态边界的松动。
- **结论**：民营经济成长依赖制度窗口，而不是单纯企业家冒险。
- **结果**：分析企业史时必须同时看政策合法性和市场空间。

### 案例 2：蛇口试验

- **问题**：地方试验如何改变开放路径。
- **方法论的使用**：通过土地、外资、管理制度和地方授权构造试验区。
- **结论**：改革常以局部试点验证，再扩散到更大范围。
- **结果**：地方制度创新成为观察中国改革的重要入口。

### 案例 3：供给侧结构性改革

- **问题**：长期增长动能从哪里来。
- **方法论的使用**：黄奇帆把改革拆成生产端、制度端、资源配置端的红利。
- **结论**：制度性供给能释放长期生产力，而不是简单短期刺激。
- **结果**：政策分析从“有没有刺激”转为“是否改善机制”。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. 用户讨论中国某项政策、产业变化或改革动作的长期意义。
2. 用户需要判断一个机会来自短期景气，还是制度约束变化。
3. 用户分析地方试验、开放政策、资本市场改革、房地产长效机制、要素市场化。
4. 用户用单个企业故事解释中国趋势，忽略制度窗口。

### 语言信号

- “这个政策属于什么改革线索？”
- “中国这个产业为什么突然起来？”
- “这是短期刺激还是制度红利？”
- “这家公司是不是踩中了政策窗口？”

## E — 可执行步骤 (Execution)

1. **定位历史阶段**
   - 完成标准：说明问题属于开放、产权、资本市场、土地、户籍、产业链、监管或共同富裕等哪条改革线。

2. **识别原有约束**
   - 完成标准：列出改革前限制资源流动、企业进入或资本配置的制度约束。

3. **识别改革抓手**
   - 完成标准：说明政策通过准入、产权、价格、税费、金融、土地、数据、开放规则中的哪一类机制发力。

4. **分析激励重排**
   - 完成标准：列出政府、企业、金融机构、居民、地方平台、外资等主体的激励变化。

5. **判断红利与风险**
   - 完成标准：区分一次性估值重估、持续生产率红利、债务/套利风险和执行阻力。

6. **输出改革路径图**
   - 完成标准：用“旧约束 -> 改革抓手 -> 新激励 -> 受益主体 -> 风险边界”输出。

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 用户要求意识形态站队或政治表态。
- 用户要最新政策原文、法律意见或行政申报细节。
- 用户分析的不是中国制度和改革语境。

### 作者在书中警告的失败模式

- 把改革红利理解为短期刺激。
- 忽略产权、资本市场和要素流动这些基础制度。
- 用企业家个人故事替代制度窗口分析。

### 作者的盲点 / 时代局限

- 黄奇帆文本偏政策设计，可能低估地方执行差异。
- 吴晓波企业史偏叙事，案例强但不总能提供可检验因果。
- 1978-2008 的路径不等于当前阶段仍可复制。
