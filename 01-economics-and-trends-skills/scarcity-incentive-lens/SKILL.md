# 稀缺、机会成本、边际和激励透镜

## R — 原文 (Reading)

> “一种东西的机会成本是为了得到这种东西所放弃的东西。”
>
> “只有一种行动的边际收益大于边际成本，一个理性决策者才会采取这项行动。”
>
> 来源：曼昆《经济学原理》第一章。

## I — 方法论骨架 (Interpretation)

这个 skill 把模糊的“值不值、该不该、好不好”改写成四个经济学问题。

第一，资源是稀缺的，任何行动都在和另一个行动竞争。
第二，真实成本不是账单，而是被放弃的最好替代选择。
第三，不要用平均值做增量决策；看下一单位行动带来的额外收益和额外成本。
第四，规则和价格一变，人会改变行为，政策和商业动作都要检查激励副作用。

它适合做第一层拆解：先把问题变清楚，再决定是否进入市场、宏观或危机分析。

## A1 — 书中的应用 (Past Application)

### 案例 1：上大学的机会成本

- **问题**：是否多读一年书。
- **方法论的使用**：不只看学费，还看放弃的工作收入和其他机会。
- **结论**：读书是否值得取决于额外收益是否超过全部机会成本。
- **结果**：把“教育好不好”转成可比较的取舍。

### 案例 2：航空公司等退票定价

- **问题**：飞机有空座时，低价卖票是否亏本。
- **方法论的使用**：区分平均成本和新增一名乘客的边际成本。
- **结论**：只要票价高于边际成本，卖出空座可能是理性的。
- **结果**：避免用平均成本误杀增量机会。

### 案例 3：安全带政策的激励反应

- **问题**：安全法规是否必然降低全部事故伤害。
- **方法论的使用**：检查驾驶者是否因更安全而开得更快或更不谨慎。
- **结论**：政策可能减少单次事故死亡，也可能增加事故次数。
- **结果**：提醒 agent 分析政策时检查间接行为反应。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. 用户在比较两个方案，问“该不该做”“是否划算”“投入值不值”。
2. 用户只列出显性成本，忽略时间、注意力、组织机会或替代方案。
3. 用户讨论政策、制度、补贴、KPI 或价格变化，但还没有检查行为反应。
4. 用户把平均成本、沉没成本或总成本当成增量决策依据。

### 语言信号

- “这个选择值不值？”
- “要不要继续投入？”
- “政策会不会产生副作用？”
- “我们已经花了这么多，要不要接着做？”

### 与相邻 skill 的区分

- 与 `market-policy-incidence` 的区别：本 skill 做基础取舍和激励拆解；市场均衡、弹性和税负归宿交给后者。
- 与 `macro-constraint-map` 的区别：本 skill 不处理国家宏观变量和制度传导。

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行：

1. **列出选择集**
   - 完成标准：至少列出当前方案、最好替代方案和“不行动”选项。

2. **重算机会成本**
   - 完成标准：把金钱、时间、注意力、资源占用、组织能力和被放弃机会分别列出。
   - 判停条件：如果用户没有给出替代方案，先要求补充或声明假设。

3. **只比较边际量**
   - 完成标准：明确“下一步行动”带来的增量收益、增量成本和主要不确定性。

4. **检查激励反应**
   - 完成标准：列出至少 3 类相关主体，并说明规则改变后他们可能如何调整行为。

5. **输出判断**
   - 完成标准：给出“支持 / 反对 / 条件成立才支持 / 信息不足”，并标明最敏感的假设。

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 用户只是查询事实、定义、数据或实时信息。
- 问题已经明确需要供需图、价格弹性、税负归宿或福利分析，应转 `market-policy-incidence`。
- 用户要求宏观预测、股价预测、汇率判断或投资建议。

### 作者在书中警告的失败模式

- 只看直接影响，不看激励变化后的间接影响。
- 用平均成本替代边际成本。
- 忽略被放弃的机会，把显性成本当作全部成本。

### 作者的盲点 / 时代局限

- 教材案例较简化，现实中的权力、制度和信息不对称可能改变激励反应。
- 对公共政策的分析不能只停留在效率，还要另行处理公平、执行和政治约束。

### 容易混淆的邻近方法论

- “沉没成本分析”：本 skill 会识别沉没成本，但不单独处理项目退出治理。
- “成本收益分析”：本 skill 是更基础的经济学透镜，不要求完整货币化。
