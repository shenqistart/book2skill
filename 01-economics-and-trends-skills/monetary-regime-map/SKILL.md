# 货币制度、央行工具和信誉地图

## R — 原文 (Reading)

> “货币政策不是灵丹妙药，但它很重要，而且非常重要。”
>
> “美元和黄金挂钩，其他货币再和美元挂钩。”
>
> 来源：伯南克《21世纪货币政策》；沃尔克、行天丰雄《时运变迁》。

## I — 方法论骨架 (Interpretation)

货币政策要同时看制度、工具、传导和信誉。

制度层问：货币价值靠什么锚定，汇率制度如何安排，央行是否独立。
工具层问：利率、资产购买、前瞻指引、流动性工具和监管职责如何组合。
传导层问：银行、市场、企业和居民是否会把央行动作转成信贷、投资和消费。
信誉层问：市场是否相信央行会坚持目标，财政和政治是否配合。

这个 skill 不预测下一步政策，而是解释央行动作为何有效、何时失灵。

## A1 — 书中的应用 (Past Application)

### 案例 1：布雷顿森林和美元锚

- **问题**：战后国际货币体系如何稳定汇率。
- **方法论的使用**：用美元兑黄金、其他货币兑美元的制度安排建立锚。
- **结论**：单一国家货币承担世界储备会内含张力。
- **结果**：特里芬难题和美元脱钩成为制度分析重点。

### 案例 2：伯南克的 QE 和前瞻性指引

- **问题**：零利率附近央行还能做什么。
- **方法论的使用**：通过资产购买压低长期利率，通过沟通影响预期。
- **结论**：央行工具箱不只短端政策利率。
- **结果**：危机后货币政策框架发生变化。

### 案例 3：通胀预期和央行信誉

- **问题**：为什么同样的政策语言影响不同。
- **方法论的使用**：检查央行过往信誉、目标一致性和财政配合。
- **结论**：预期稳定依赖可信承诺，而非单次讲话。
- **结果**：货币政策的艺术部分来自信誉管理。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. 用户问央行降息、加息、缩表、扩表、QE、前瞻性指引为何起作用。
2. 用户讨论美元体系、人民币国际化、汇率锚或资本流动。
3. 用户想理解通胀预期、央行信誉和政策沟通。
4. 用户把货币政策当成能解决所有宏观问题的单一按钮。

- “QE 到底怎么影响经济？”

## E — 可执行步骤 (Execution)

1. **识别货币制度**
   - 完成标准：说明汇率安排、货币锚、央行目标和独立性。

2. **列出政策工具**
   - 完成标准：区分利率、资产负债表、流动性工具、沟通工具、监管工具。

3. **检查传导渠道**
   - 完成标准：说明政策如何通过银行、债券、资产价格、汇率、预期传到实体经济。

4. **评估信誉和配合**
   - 完成标准：检查央行目标一致性、财政政策配合、监管职责和市场预期。

5. **输出制度判断**
   - 完成标准：用“可能有效的部分、可能堵塞的部分、依赖假设、不能判断的实时事项”输出。

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 用户要求预测议息会议、汇率点位、债券收益率或交易建议。
- 用户需要最新央行公告、实时通胀数据或监管条文。
- 用户讨论的不是货币、金融稳定或宏观传导。

### 作者在书中警告的失败模式

- 把货币政策当作灵丹妙药。
- 忽略央行信誉和预期管理。
- 忽略金融监管不到位会削弱金融稳定。

### 作者的盲点 / 时代局限

- 美国央行经验不能直接套到资本账户、金融结构和财政关系不同的经济体。
- 危机工具在非常时期有效，不代表常态下没有副作用。
