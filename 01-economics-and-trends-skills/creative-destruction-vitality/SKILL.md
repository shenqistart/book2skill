# 创造性破坏与经济活力诊断

## R — 原文 (Reading)

> “生产力是衡量经济成功与否的终极指标。”
>
> “创造与破坏是一对孪生兄弟。”
>
> 来源：格林斯潘、伍尔德里奇《繁荣与衰退》引言。

## I — 方法论骨架 (Interpretation)

长期活力不等于热闹、融资多或企业数量多。

它要看新技术、新组织和新企业是否把资源从低生产率用途转到高生产率用途。
创造性破坏有两面：创造新产出、新岗位、新组织，也摧毁旧资产、旧岗位和旧秩序。
一个经济体是否有活力，取决于进入是否容易、失败是否可承受、规则是否允许资源重组、社会是否能处理调整成本。

这个 skill 用来判断增长是不是由生产率驱动，而不是由泡沫、补贴或叙事驱动。

## A1 — 书中的应用 (Past Application)

### 案例 1：美国工业化和企业组织

- **问题**：美国为何能形成高生产率增长。
- **方法论的使用**：观察钢铁、铁路、能源、企业组织、产权和资本市场如何重组资源。
- **结论**：生产率来自技术、组织和市场制度的组合。
- **结果**：经济史分析聚焦生产率，而不只看 GDP 数字。

### 案例 2：反托拉斯和政治修正

- **问题**：创造性破坏为何引发监管反弹。
- **方法论的使用**：观察巨型企业、财富集中、社会愤怒和政治回应。
- **结论**：活力需要规则修正，否则破坏成本会侵蚀合法性。
- **结果**：长期活力必须包含社会承受力。

### 案例 3：中国企业从制度缝隙中成长

- **问题**：中国企业为什么在非规范市场中高速成长。
- **方法论的使用**：吴晓波记录民营、国营、外资三股力量的博弈。
- **结论**：企业活力来自制度松动、市场需求和企业家行动的共同作用。
- **结果**：不能把企业成长只归功于个人能力或单一政策。

## A2 — 触发场景 (Future Trigger)

### 用户会在什么情境下需要这个 skill?

1. 用户问某行业、城市或经济体是否还有长期增长活力。
2. 用户看到融资、创业、政策热潮，想判断是否真的提升生产率。
3. 用户讨论创新、颠覆、平台、反垄断、监管负担或企业退出。
4. 用户把“破坏旧行业”本身当作进步。

### 语言信号

- “这个行业是真繁荣还是泡沫？”
- “怎么判断一个地方有没有经济活力？”
- “创新会不会只是烧钱？”
- “监管会不会扼杀活力？”

### 与相邻 skill 的区分

- 与 `industrial-policy-discipline` 的区别：本 skill 看长期活力和进入退出；后者看产业政策设计是否有纪律。
- 与 `china-reform-path` 的区别：本 skill 可用于一般经济体；中国制度阶段分析先用后者。

## E — 可执行步骤 (Execution)

1. **识别生产率来源**
   - 完成标准：判断增长来自技术、组织、规模、资本深化、资源重配、需求扩张、补贴或资产价格。

2. **检查进入与退出**
   - 完成标准：说明新企业能否进入，低效企业能否退出，失败成本由谁承担。

3. **检查组织和资本配置**
   - 完成标准：判断资本是否流向高生产率用途，企业组织是否提升协调效率。

4. **检查破坏成本**
   - 完成标准：列出旧岗位、旧资产、地方财政、社会稳定和监管反弹风险。

5. **判断制度承受力**
   - 完成标准：说明规则是否允许竞争、破产、兼并、再培训和社会补偿。

6. **输出活力诊断**
   - 完成标准：给出“真实活力 / 叙事繁荣 / 补贴驱动 / 泡沫驱动 / 信息不足”，并说明证据。

## B — 边界 (Boundary)

### 不要在以下情况使用此 skill

- 用户要求短期股价、融资估值或行业景气预测。
- 用户只想为“颠覆旧行业”寻找道德辩护。
- 缺少生产率、进入退出、资本配置或制度环境信息。

### 作者在书中警告的失败模式

- 只看创造，不看破坏。
- 只看 GDP 或企业规模，不看劳动生产率和全要素生产率。
- 让监管僵化到阻碍进入，或让垄断强到阻止新竞争。

### 作者的盲点 / 时代局限

- 美国经验强调产权和企业家精神，未必能解释所有发展型经济体。
- 中国企业史案例带有阶段性，不能直接推出当前行业结论。
