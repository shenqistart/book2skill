# 候选池：术语

| 术语 | 来源 | 蒸馏用法 |
|---|---|---|
| 机会成本 | 曼昆 | 用放弃的次优选择衡量成本 |
| 边际变动 | 曼昆 | 只比较增量收益和增量成本 |
| 激励 | 曼昆 | 规则改变后行为主体的反应机制 |
| 弹性 | 曼昆 | 需求或供给对条件变化的敏感度 |
| 无谓损失 | 曼昆 | 政策或税收减少交易后损失的总剩余 |
| 预算软约束 | 徐高 | 主体预期会被救助，导致约束失真 |
| 货币锚 | 黄奇帆、沃尔克 | 货币发行和价值稳定的制度基础 |
| 前瞻性指引 | 伯南克 | 央行用沟通影响市场预期的工具 |
| 最后贷款人 | 伯南克 | 中央银行在恐慌中提供流动性支持 |
| 监管钟摆 | 布林德 | 危机前宽松、危机后过紧的监管周期 |
| 出口纪律 | Studwell | 以国际市场表现检验扶持政策 |
| 供给侧结构性改革 | 黄奇帆 | 通过体制机制改革释放长期生产力 |
| 创造性破坏 | 格林斯潘/伍尔德里奇 | 新技术和组织替代旧资产并提升生产率 |
