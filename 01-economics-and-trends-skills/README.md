# 01 经济学和大势 Skills

本目录把产品分类 `01_经济学和大势` 下的经济学教材、宏观政策、货币金融、发展战略和企业史书籍，蒸馏成一组可被 agent 调用的分析 skills。

它不是书摘，也不是投资建议。它的用途是：当用户面对政策、市场、产业、金融风险或商业大势问题时，先选对分析框架，再按步骤拆解，不把宏观叙事当成万能解释。

## 推荐调用方式

1. 先读 [INDEX.md](./INDEX.md)，按用户问题选择 skill。
2. 如果问题是基础决策或市场机制，优先从 `scarcity-incentive-lens` 或 `market-policy-incidence` 开始。
3. 如果问题涉及中国宏观、改革、房地产、产业链，优先从 `macro-constraint-map`、`china-reform-path`、`industrial-policy-discipline` 中选择。
4. 如果问题涉及央行、汇率、金融危机，优先使用 `monetary-regime-map` 或 `financial-crisis-forensics`。
5. 如果问题是判断长期经济活力或企业史位置，使用 `creative-destruction-vitality`。

## 重要边界

- 不提供证券买卖建议、收益预测、政策押注或宏观择时。
- 不把美国经验、中国经验或东亚经验直接互套。
- 不把书中的历史判断当成当前事实；涉及实时政策、数据和监管时需要另行查证。
- 不复制原书大段内容，源文本只用于方法论审计。
