# 被淘汰候选

## R01 “要重视经济大势”

- **不通过项**：V3 独特性。
- **原因**：表述过宽，任何宏观读书笔记都会说，不能指导 agent 执行。
- **处理**：降级为 README 说明，不做 skill。

## R02 “中国企业家很重要”

- **不通过项**：V2 预测力、V3 独特性。
- **原因**：如果不补充制度窗口、产权约束、资本条件和行业结构，只是常识性判断。
- **处理**：企业家因素并入 `china-reform-path` 和 `creative-destruction-vitality`。

## R03 “美元霸权正在变化”

- **不通过项**：V2 预测力。
- **原因**：容易变成宏观判断或预测口号，缺少可执行步骤。
- **处理**：将其中可执行部分并入 `monetary-regime-map` 的货币锚和汇率制度分析。

## R04 “房地产很关键”

- **不通过项**：V3 独特性。
- **原因**：单独作为结论过于常识，且容易诱发当前价格预测。
- **处理**：放入 `macro-constraint-map` 和 `china-reform-path` 的约束项。

## R05 “历史会周期性重演”

- **不通过项**：V1 跨域、V2 预测力。
- **原因**：缺少明确机制，容易用相似故事代替因果分析。
- **处理**：历史案例只作为 A1，不独立成 skill。
