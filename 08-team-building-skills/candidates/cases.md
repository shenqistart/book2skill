# 候选案例

## OKR 组织案例

- Intel/Google 系 OKR 强调少数目标、公开对齐、评分和复盘。
- 案例价值：支持 `okr-quality-design` 与 `okr-alignment-review-cadence`。
- 抽取边界：不复制具体公司目标，只抽象设计与运行原则。

## Netflix 文化案例

- 《奈飞文化手册》反复把高绩效团队、成年人假设、坦诚沟通、去流程化和岗位适配放在一起。
- 案例价值：支持 `freedom-responsibility-culture` 与 `talent-density-role-fit`。
- 抽取边界：Netflix 的制度不是通用答案，只有在高信任、高信息透明、高人才密度环境下才适用。

## 个人效能案例

- 《创造时间》把每天的重点、注意力防护、能量和复盘连成小循环。
- 《七个习惯》把个人主动性、影响圈、要事优先和互赖协作连成更长期的行为系统。
- 案例价值：支持个人到团队执行的桥接。

## 结构化表达案例

- 《金字塔原理》从读者问题、引言情境、分组逻辑和结论先行构造表达。
- 案例价值：支持把 OKR、文化变革、人才判断写成可讨论材料。
