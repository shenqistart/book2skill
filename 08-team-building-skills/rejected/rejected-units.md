# 被淘汰候选

## R1: 个人成功语录集合

- 来源：《穷查理宝典》《七个习惯》
- 淘汰原因：容易变成励志摘抄，无法形成稳定触发、步骤和可验证输出。

## R2: 复制 Google/Intel/Netflix 的制度清单

- 来源：两本 OKR 书、《奈飞文化手册》
- 淘汰原因：组织上下文差异过大；本仓库只保留方法论，不复制公司制度。

## R3: 纯时间管理技巧大全

- 来源：《创造时间》
- 淘汰原因：书中有大量具体战术，但团队建设场景更需要“每日重点循环”这个核心结构。

## R4: 金字塔模板大全

- 来源：《金字塔原理》
- 淘汰原因：模板多但容易机械化；保留“读者问题 -> 结论先行 -> 逻辑分组 -> 证据检查”的核心 skill。

## R5: 领导者人格画像

- 来源：《七个习惯》《奈飞文化手册》
- 淘汰原因：描述性强、可执行性弱，且容易误导 agent 做性格评判。

## R6: 投资判断 skill

- 来源：《穷查理宝典》
- 淘汰原因：本分类为团队建设，不处理证券投资建议；只保留决策质量和误判防护。
