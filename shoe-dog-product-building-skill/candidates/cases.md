# 候选方法论单元

- id: field-market-validation
  title: 真实场地市场验证
  type: framework
  source_chapter: 1962-1966
  summary: |
    当用户有一个产品想法但还停在桌面推演，需要到真实使用场地验证需求、销售和反馈时调用。不适用于纯线上问卷汇总。
  V1_cross_domain: true
  V2_predictive_power: true
  V3_exclusivity: true
  tags: [startup, validation, product]

- id: athlete-feedback-product-loop
  title: 运动员反馈产品循环
  type: framework
  source_chapter: 鲍尔曼实验、普雷方丹与早期跑鞋
  summary: |
    当产品强依赖真实高强度使用体验，需要把专业用户、试穿、材料实验和快速改进连成循环时调用。不适用于低风险装饰性改动。
  V1_cross_domain: true
  V2_predictive_power: true
  V3_exclusivity: true
  tags: [product, feedback, sports]

- id: supplier-dependence-risk
  title: 供应依赖风险拆解
  type: framework
  source_chapter: 1970-1972 鬼冢虎危机与 Nike 诞生
  summary: |
    当产品业务依赖单一供应商、渠道、平台或授权方，且对方可能改变条件时调用。不适用于已经有多源稳定供应的小采购问题。
  V1_cross_domain: true
  V2_predictive_power: true
  V3_exclusivity: true
  tags: [supply-chain, risk, startup]

- id: cashflow-growth-discipline
  title: 增长现金流纪律
  type: framework
  source_chapter: 1970、1975、1980 上市
  summary: |
    当业务销售增长很快但现金、库存、银行信用和供应账期紧绷时调用。不适用于已经现金流充裕的品牌营销讨论。
  V1_cross_domain: true
  V2_predictive_power: true
  V3_exclusivity: true
  tags: [cashflow, growth, startup]

- id: brand-movement-building
  title: 品牌运动构建
  type: framework
  source_chapter: Nike 命名、Swoosh、运动员与广告
  summary: |
    当用户想把产品从商品做成一种运动、态度或社群身份，而不是只做广告时调用。不适用于短期促销文案生成。
  V1_cross_domain: true
  V2_predictive_power: true
  V3_exclusivity: true
  tags: [brand, community, product]
