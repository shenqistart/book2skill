    # 《简约至上》Skill 索引

    > 本书由 book2skill / cangjie-skill 蒸馏，产出 **6** 个 skills。处理时间：2026-06-18

    ## 关于这本书

    - **作者**: Giles Colborne
    - **出版年**: 2018 中文第 2 版
    - **一句话主旨**: 用愿景、主流用户和删除、组织、隐藏、转移四策略，把复杂交互变成真正简单而不是冒充简单。
    - **整书理解**: [BOOK_OVERVIEW.md](BOOK_OVERVIEW.md)

    ## Skill 列表

    | Skill | 中文名 | 什么时候用 |
    |---|---|---|
    | [`simplicity-vision-blueprint`](simplicity-vision-blueprint/SKILL.md) | 简单愿景蓝图 | 当团队想简化产品但不知道核心用户、品牌特征和体验愿景是什么时调用。 |
| [`delete-strategy`](delete-strategy/SKILL.md) | 删除策略 | 当界面、流程或产品被非必要功能、文字、选项和错误负担拖慢，需要判断能删什么时调用。 |
| [`organize-strategy`](organize-strategy/SKILL.md) | 组织策略 | 当信息或功能不能删除，但需要通过分块、行为、层级、位置和模式降低认知负担时调用。 |
| [`hide-strategy`](hide-strategy/SKILL.md) | 隐藏策略 | 当功能不常用但不能删除，需要决定何时、何处、如何渐进展示时调用。 |
| [`transfer-strategy`](transfer-strategy/SKILL.md) | 转移策略 | 当复杂性无法删除、组织或隐藏，需要转移到设备、系统、用户、开放体验或可信机制中时调用。 |
| [`mainstream-user-focus`](mainstream-user-focus/SKILL.md) | 主流用户聚焦 | 当专家用户、高声量用户或内部人员要求增加复杂功能，团队需要回到主流用户体验时调用。 |
