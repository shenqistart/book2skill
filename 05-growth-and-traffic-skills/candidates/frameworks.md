# Framework Candidates

- id: f01
  title: 回头客体验复利
  type: framework
  source_chapter: 《回头客战略》成长的平台：品牌、文化、培训
  source_quote: |
    "把本应投到广告上的资金运用到提高客户服务和顾客体验上。"
  summary: |
    把流量问题改写为体验、复购和口碑问题。企业先把服务、物流、退换货、文化和培训做成用户愿意主动传播的体验，再让回头客降低长期获客压力。
  tags: [retention, customer-experience, word-of-mouth]

- id: f02
  title: 啊哈与不可或缺性门禁
  type: framework
  source_chapter: 《增长黑客》第二章 好产品是增长的根本
  source_quote: |
    "40%或者更多的回复是'非常失望'，就意味着产品已经获得了足够的不可或缺性。"
  summary: |
    在增长攻势前，先确认用户是否体验到核心价值。用不可或缺性调查、关键行为和留存曲线判断是否具备增长资格。
  tags: [aha, product-market-fit, retention]

- id: f03
  title: 增长实验操作系统
  type: framework
  source_chapter: 《增长黑客》第一章；《硅谷增长黑客实战笔记》第一章
  source_quote: |
    "通过迅速测试新想法、新思路，并根据计划指标对结果进行评估。"
  summary: |
    将增长从项目制营销改为跨职能实验系统。团队围绕一个目标提出假设、排序、最小测试、分析结果，并把学习反馈进下一轮实验。
  tags: [growth-team, experiment, operating-system]

- id: f04
  title: 北极星指标与增长模型
  type: framework
  source_chapter: 《硅谷增长黑客实战笔记》第二章
  source_quote: |
    "北极星指标就像北极星一样，高高闪耀在空中。"
  summary: |
    先定义代表用户获得核心价值的北极星指标，再把它拆成用户旅程、输入变量和增长仪表盘，用模型统一优先级。
  tags: [north-star, growth-model, metrics]

- id: f05
  title: OMTM 精益数据聚焦
  type: framework
  source_chapter: 《精益数据分析》第六章
  source_quote: |
    "挑选一个唯一的指标，该指标对你当前所处的创业阶段无比重要。"
  summary: |
    根据商业模式和阶段选择唯一最重要指标，用可行动指标、同期群和底线判断下一步是坚持、转型、加速还是刹车。
  tags: [omtm, lean-analytics, cohort]

- id: f06
  title: 靶心渠道选择
  type: framework
  source_chapter: 《拉新》第三、四章
  source_quote: |
    "靶心对所有类型的创业公司都适用。"
  summary: |
    把 19 类渠道放进外环、中环和内环。先广泛构想，再低成本测试几个最有潜力渠道，最后集中资源打穿一个核心渠道。
  tags: [traction, acquisition, channel]

- id: f07
  title: 私域关系飞轮
  type: framework
  source_chapter: 《关系飞轮》第三、四章
  source_quote: |
    "私域就是那些你直接拥有、可重复、低成本甚至免费触达的用户。"
  summary: |
    私域不是触达工具，而是在一对多中模拟一对一亲密关系。通过可信、可爱、可亲三种关系，让普通用户升级为超级用户。
  tags: [private-domain, relationship, super-user]

- id: f08
  title: 小群社群杠杆
  type: framework
  source_chapter: 《小群效应》第一、二、三章
  source_quote: |
    "大群松散沉默，小群紧密活跃。"
  summary: |
    社群价值来自相互认识、相互信赖和频繁互动。用三近一反、连接者和工具性场景，让小群承担扩散、转化和留存。
  tags: [community, small-group, connector]

- id: f09
  title: 直播电商适配地图
  type: framework
  source_chapter: 《直播电商的逻辑》第一、二、三章
  source_quote: |
    "直播电商并没有改变人、货、场匹配的零售基本逻辑。"
  summary: |
    判断品类是否适合直播，应同时检查商品展示难度、主播信任、平台算法、供应链履约、价格体系和售后体验。
  tags: [live-commerce, platform, retail]

- id: f10
  title: 平台权力迁移策略
  type: framework
  source_chapter: 《流媒体时代》第一、七、八章
  source_quote: |
    "这些数据使得特德·萨兰多斯相信，《纸牌屋》的创意完全行得通。"
  summary: |
