# 《茶经》候选方法论单元

> 源文件: [../source/chajing.md](../source/chajing.md)

## 通过候选

1. **茶叶品质与风险判断**
   - 来源章节: 一之源、三之造、八之出
   - 价值: 把产地、采摘形态、制作状态和风险合并为诊断框架。
   - 通过原因: 多章反复出现，能外推到茶样描述分析。

2. **制茶流程检查**
   - 来源章节: 二之具、三之造、九之略
   - 价值: 把采、蒸、捣、拍、焙、穿、封变成流程审查。
   - 通过原因: 具备明确顺序和失败条件。

3. **工具与器具系统**
   - 来源章节: 二之具、四之器、九之略
   - 价值: 区分制茶工具、煮饮器、场景取舍。
   - 通过原因: 器具不是名物罗列，而是功能系统。

4. **水火煮茶法**
   - 来源章节: 五之煮、六之饮
   - 价值: 适合转成可执行的煮茶步骤。
   - 通过原因: 火、水、沸候、投茶、分饮均有判断点。

5. **品饮场景安排**
   - 来源章节: 六之饮、十之图
   - 价值: 处理人数、盏数、次第和饮用方式。
   - 通过原因: 有明确数量和流程边界。

6. **场景化简化取舍**
   - 来源章节: 九之略
   - 价值: 决定在野外、山寺、城邑等环境哪些可省。
   - 通过原因: 是全书最直接的条件化规则。
